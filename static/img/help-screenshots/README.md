# Help Page Screenshots

This folder contains screenshots for the help page, organized by language.

## Directory Structure

- `/en/` - English screenshots
- `/it/` - Italian screenshots
- (Add more language folders as needed: `/es/`, `/fr/`, `/de/`)

## Image Files (per language folder)

1. **01-search-by-title-or-artist.png** - Shows the search box with example search query
2. **02-click-song-to-request.png** - Shows clicking on a song card to request it
3. **03-clear-search.png** - Shows the X icon to clear the search
4. **04-view-your-requests.png** - Shows the "Your Requests" section with requested songs
5. **05-remove-requests.png** - Shows the remove button (X icon) next to requested songs
6. **06-reset-search.png** - Shows resetting/clearing the search

## How to Replace Images

Simply replace the image files with new ones, keeping the same filenames. The images will automatically update on the help page based on the user's selected language.

## Fallback Behavior

If an image is not found for the current language, the system will automatically fall back to the English version (`/en/` folder).

## Image Specifications

- Format: PNG (recommended) or JPG
- Recommended size: 500-800px width
- Aspect ratio: Maintain original proportions
- File size: Keep under 500KB for fast loading
